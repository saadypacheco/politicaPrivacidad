Política de Privacidad – AppAgentesn8n

Última actualización: 04/12/2025

La aplicación AppAgentesn8n se utiliza exclusivamente para integrar automatizaciones internas mediante la plataforma n8n. No está destinada al uso público ni se comercializa.

1. Datos que recopilamos

La aplicación puede acceder únicamente a la información necesaria para:

  • Autenticar al usuario mediante Facebook.
  • Ejecutar automatizaciones internas autorizadas por el propietario de la aplicación.

No se recopila información adicional ni se comparte con terceros.

2. Uso de los datos

Los datos obtenidos se utilizan exclusivamente para:

  • Vincular la cuenta de Facebook con el sistema n8n.
  • Ejecutar flujos internos configurados por el administrador.

No se venden, distribuyen ni transfieren datos a ningún tercero.

3. Eliminación de datos

El usuario puede solicitar la eliminación de sus datos enviando un correo a:

Email: saadypacheco@gmail.com

Una vez recibida la solicitud, se eliminará de forma permanente toda la información almacenada vinculada al usuario y se confirmará la eliminación por correo electrónico.

4. Contacto

Para consultas sobre esta política o sobre el uso de datos, contactarse a:

Email: saadypacheco@gmail.com
